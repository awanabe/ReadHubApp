# readhubapp

> My amazing app


## Dev

```
$ ELECTRON_MIRROR="https://npm.taobao.org/mirrors/electron/" npm install
```

### Run

```
$ npm start
```

### Build

```
$ npm run build
```

Builds the app for macOS, Linux, and Windows, using [electron-packager](https://github.com/electron-userland/electron-packager).
